Getting Started With Flexify

Before start I recommend use reset.css or similar library for more consistent results. Choose one of the variants and include it in your project.

Full version

Flexify full version include all flex utilities for flexboxes (as clases) with breakpoints for better control in responsive websites (based on bootstrap breakpoints).

lite

Lite flexify include all basic utilities ut doesn't include responsive breakpoints so you will have to do manually any variant for responsive pages.

lite div-forced

Includes the same as the lite version but this version change the display property of the divs to flex so you don't have to add d-flex classes on all divs

Docs

Flexify its a CSS library that include basic utilities to create layouts based on flexboxes.

There are not active utilities for flex-basis, flex-shrink and flex-grow properties. You can (sholud) add manually these properites to your flex components. You can check faq for more info

.d-flex

set display: flex to the element. The default direction is row

.flex-column

Changes the direction to column

.justify-center

set justify-content: center property

.justify-start

set justify-content: flex-start property

.justify-end

set justify-content: flex-end property

.justify-space-around

set justify-content: space-around property

.justify-space-between

set justify-content: space-between property

.justify-unset

set justify-content: unset property

.align-center

set align-items: center property

.align-start

set align-items: flex-start property

.align-end

set align-items: flex-end property

.align-unset

set align-items: unset property
